# C/C++学习笔记

## [align.c](./align.c)

根据输入参数对齐输入文本。

```bash
$ gcc -g align.c
$ ./a.out 80 < input.log
```

### 算法设计到一半就动手

在搞清楚自己想用的算法之前，不要动手写程序。

### 以为argc是第一个参数

argc是命令中所有参数的个数（包括./a.out）。

### 没有输出最后一个单词

主循环结束后还会剩一个单词没有输出。

### 输入文本中使用了非ascii字符

从别处复制过来的文本里包括了非ascii字符，这使得某一行计算字符数的时候总会多加一些。把那个字符换成对应的ascii字符就好了。
